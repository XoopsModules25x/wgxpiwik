Help: wgXPiwik

Description

The XOOPS module wgXPiwik you can use for implementation of Piwik into your XOOPS-CMS

Install/uninstall

No special measures necessary, follow the standard installation process and extract the wgxpiwik folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Features

Basics:

  • Piwik is included in this module
  • You can install Piwik from this module
  • You can not manage the settings of Piwik himself (create users, manage user right,...). You have to us Piwik UI.
  • You can provide the Piwik Charts via iframe on user side. Because of using the Piwik function 'logme' there is no seperate login necessary.
  • You can manage permissions for each xoops uer group
  • Tracking your sites is done with a block. You can activate tracking for specific sites, all sites or only frontpage.
